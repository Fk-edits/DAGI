🌍 1. World Wide Web & Design Principles

core theory
📡 The Web vs Internet
<!-- Quick facts -->
Internet = global network of computers.
WWW   = service (webpages) over HTTP.
Foundations: HTML, HTTP, Browser, Server.

✅ Key takeaway: Websites run on the Web. E‑gov, e‑learning, e‑commerce all use websites.

🎯 Design guidelines: target audience, navigation, consistency, simplicity, accessibility.

📘 Concept The Web is an information system — Tim Berners-Lee, 1989. Websites = collection of pages. Good design = better UX.

🏛️ 2. HTML Document Structure

<!DOCTYPE html>
📄 Minimal HTML skeleton
<!DOCTYPE html>
<html>
<head><title>My Page</title></head>
<body>
  <h1>Hello Dagim</h1>
  <p>This is a paragraph.</p>
</body>
</html>

Hello Dagim

This is a paragraph.

<html> root element  |  <head> metadata  |  <body> visible content  |  <h1> main heading  |  <p> paragraph

🔗 5. Hyperlinks — Absolute, Relative, Image & Section Links

<a> anchor
🌐 External links & target attribute
<a href="https://www.ethiotelecom.et" target="_blank">Ethio Telecom (new tab)</a>
<a href="https://www.cbe.com.et">Commercial Bank of Ethiopia</a>
📡 Ethio Telecom (new tab)
🏦 Commercial Bank of Ethiopia
href destination URL • target="_blank" open new tab • default _self
🖼️ Image as a clickable link
<a href="https://www.ethiopianairlines.com">
  <img src="https://picsum.photos/id/13/150/80" alt="Ethiopian Airlines">
</a>

click image → Ethiopian Airlines website

🔹 Nesting <img> inside <a> creates image hyperlink.
📍 Internal link (same page section)
<a href="#pro-tips-section">Go to Pro Tips</a>
...
<h3 id="pro-tips-section">🔥 Pro Tips</h3>
⬇️ Click to jump to Pro Tips section (at bottom)

*smooth scroll demo - see footer area

id attribute creates destination, href="#id" links to it. Perfect for table of contents.
🌐 Absolute vs Relative URL
<!-- Absolute -->
https://www.example.com/images/logo.png
<!-- Relative (assuming same folder) -->
<a href="about.html">About</a>
<img src="assets/photo.jpg">
Absolute = full path (protocol+domain)
Relative = path relative to current file
✅ Best practice: use relative for internal resources.

📊 6. HTML Tables — Data Grids, Merging Cells

<table> <tr> <td> <th>
📋 Basic student marks table
<table border="1">
  <tr><th>Name</th><th>Score</th></tr>
  <tr><td>Dagim</td><td>98</td></tr>
  <tr><td>Betty</td><td>94</td></tr>
</table>
NameScore
Dagim98
Betty94
<table> defines table • <tr> row • <th> header cell • <td> data cell
🧩 colspan & rowspan (merge magic)
<table border="1">
  <tr><th colspan="2">Semester Report</th></tr>
  <tr><td rowspan="2">Dagim</td><td>HTML: 100</td></tr>
  <tr><td>CSS: 97</td></tr>
</table>
Semester Report
DagimHTML: 100
CSS: 97
colspan merges columns horizontally • rowspan merges rows vertically — powerful for dashboards.
🏗️ Table as page layout (classic)
<table width="100%">
  <tr bgcolor="#ccc"><td colspan="2">HEADER</td></tr>
  <tr><td width="30%">Sidebar</td><td>Content</td></tr>
  <tr><td colspan="2">Footer</td></tr>
</table>
📌 HEADER
🔹 Sidebar✨ Main content area
© Footer

📃 7. HTML Lists — Ordered, Unordered, Description & Nesting

<ol> <ul> <dl>
🔢 Ordered list (steps)
<ol type="1">
  <li>Learn HTML</li>
  <li>Practice tables</li>
  <li>Build project</li>
</ol>
  1. Learn HTML
  2. Practice tables
  3. Build project
<ol> ordered/numbered • type (1, A, a, I) • <li> list item
🔘 Unordered list (bullet)
<ul type="circle">
  <li>HTML5</li>
  <li>CSS3</li>
  <li>JavaScript</li>
</ul>
  • HTML5
  • CSS3
  • JavaScript
<ul> unordered list • bullet styles: disc, circle, square
📖 Description list (glossary)
<dl>
  <dt>HTML</dt><dd>HyperText Markup Language</dd>
  <dt>HTTP</dt><dd>HyperText Transfer Protocol</dd>
</dl>
HTML
HyperText Markup Language
HTTP
HyperText Transfer Protocol
<dl> description list • <dt> term • <dd> definition
🌲 Nested lists (mix types)
<ul>
  <li>Frontend
    <ol><li>React</li><li>Vue</li></ol>
  </li>
  <li>Backend
    <ul><li>Django</li><li>Flask</li></ul>
  </li>
</ul>
  • Frontend
    1. React
    2. Vue
  • Backend
    • Django
    • Flask
✅ Nest any list type inside another — perfect for outlines.
